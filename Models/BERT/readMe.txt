test_MT.tags.txt: This file contains tags for the associated words in the test set

test_MT.words.txt: This file contains words in the test set

train_MT.tags.txt: This file contains tags for the associated words in the training set

train_MT.words.txt: This file contains words in the training set

bert.ipynb: This is Jupyter notebook that reads in all of the files above, trains the BERT model and outputs the train and test accuracy
